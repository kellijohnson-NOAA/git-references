# Summary
Developed a multi-species age-structured population dynamics model for walleye pollock, Atka mackerel, and Pacific cod on the Aleutian Shelf, Alaska. Using AIC, seven functional responses were investigated, where the functional response acts as a measure of predator effort or the rate at which predators impose mortality on their prey. Predators impose this rate of mortality differently on different species and size classes according to a given "selectivity" or desire for a given species/age class combination given their own species/age class combination. Predator selectivity was measured using diet samples.
